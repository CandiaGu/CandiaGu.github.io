Globe

December 8th, 2016

Globe is a visually-interesting game of which the objective is for the character to catch comets and plant trees on this revolving red planet. This 2.5D game uses facial detection to sense whether the user is still playing or not and pauses the game for them if not.

This game was coded entirely in python and uses various libraries such as pygame and opencv for ease of use.
